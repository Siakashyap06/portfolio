## Option 1: Double-Click (Easiest - macOS)
1. Double-click `start-server.command`
2. Browser will open automatically
3. Server runs at: **http://localhost:5500**

## Option 2: Terminal Command
```bash
./start-server.sh
```

## Option 3: Manual (One Line)
```bash
python3 -m http.server 5500
```
Then open: http://localhost:5500

## Option 4: No Server Needed!
Just open `index.html` directly in your browser - everything works standalone!

---

**Note:** The server uses Python's built-in HTTP server (no external apps needed). It's already installed on your Mac.
